SAE S2.02 - Interface sur le réchauffement climatique

Présentation du projet

Ce projet réalisé en binôme sur 2 jours avait pour but de sensibiliser au réchauffement climatique via une interface interactive en C++. Nous avons utilisé les données réelles de températures de la station du Mont Aigoual pour créer un graphique évolutif.

Objectifs

  • Analyser les données météorologiques fournies.
  • Créer une interface graphique dynamique avec LibGraph2.
  • Ajouter une interaction utilisateur pour visualiser les décennies.
  • Illustrer les effets du réchauffement avec des personnages animés (ours).

Captures et illustrations

Techniques et développement

Le projet a été développé en C++ avec la bibliothèque LibGraph2. Deux classes principales ont été créées :

  • Donnée : pour importer et structurer les températures.
  • Diagramme : pour afficher les courbes, axes, seuils et interactions visuelles.
Nous avons également programmé une barre de navigation temporelle et des animations d’ours selon les hausses ou baisses de température.

Compétences mobilisées

  • Programmation C++ avec LibGraph2
  • Traitement et visualisation de données
  • Travail en binôme
  • Conception d’interface utilisateur
  • Créativité visuelle et logique algorithmique

Outils utilisés

  • Visual Studio Code
  • LibGraph2
  • Discord, Gmail
  • Excel (pour les données météo)

Conclusion

Ce projet a permis de combiner nos compétences techniques avec une cause importante. Il démontre l’impact que peut avoir une interface bien pensée pour illustrer des données sensibles comme le climat. Nous avons aussi pris conscience de l’importance de la coordination en binôme pour un travail efficace.
